# README

## 提交规范

- 见阿里巴巴 commit 规范[如何规范你的 Git commit？-阿里云开发者社区 (aliyun.com)](https://developer.aliyun.com/article/770277)

- 例如：

  ```
  git commit -M "chore: 初始化项目"
  git commit -M "feat: 完成用户登录部分
  ```

- 先拉取 dev，然后根据 dev 自己建立自己的分支，修改以后提交 PR 到 dev 分支上面
- 测试 1
